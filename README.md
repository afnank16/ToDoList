# ✅ To Do List App

A clean and minimal To Do List built with React + Tailwind CSS. Tasks persist across sessions using localStorage — no backend needed.

## Features

- ➕ Add tasks by typing and pressing Enter or clicking Add
- ✅ Mark tasks as complete with a checkbox
- 🗑️ Delete tasks
- 💾 Auto-saves to localStorage (persists on refresh)
- 📊 Live completion counter

## Setup

```bash
npm install
npm run dev
```

## Built With

- React
- Tailwind CSS
- localStorage API
